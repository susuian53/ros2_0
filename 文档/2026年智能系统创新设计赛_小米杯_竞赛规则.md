# 2026 年全国大学生计算机系统能力大赛

# 智能系统创新设计赛（小米杯）竞赛规则

## 一、任务及竞赛规则

### 1. 石径探路

- **赛段要求：** 机器狗摆放在起点位置，头朝箭头方向，处于趴下状态；
- **开始比赛：** 机器狗站起，开始计时；
- **赛段结束：** 后腿足底离开弯道虚线后。

**竞赛规则：**

1. 出线/踩线按评分标准扣分；
2. 跌倒或人工干预移动设备按评分标准扣分；
3. 未通过石板/弯道拐弯按评分标准扣分；
4. 本赛段不允许跳过。

### 2. 荒野寻珠

- **赛段要求：** 寻找并撞击橙色小球，可以使用身体任何部位来撞击，需要小球有明显晃动；
- **小球位置调整：** 入口处 3 个小球固定使用浅蓝色（R4C4，R4C3，R3C4），其余小球会在赛前调整，规则为每行每列都有 1 个橙色小球；
- **赛段结束：** 后腿足底离开出口虚线后。

**竞赛规则：**

1. 出线/踩线按评分标准扣分；
2. 跌倒或人工干预移动设备按评分标准扣分；
3. 触碰非目标小球按评分标准扣分；
4. 未触碰目标小球按评分标准扣分；
5. 未到达出口位置按评分标准扣分；
6. 放弃本赛段按评分标准扣分。

### 3. 曲道冲锋

- **赛段要求：** 全程在赛道内行进，不得踩线或者越线；
- **赛段结束：** 后腿足底离开弯道虚线后。

**竞赛规则：**

1. 出线/踩线按评分标准扣分；
2. 跌倒或人工干预移动设备按评分标准扣分；
3. 未到达出口位置按评分标准扣分；
4. 放弃本赛段按评分标准扣分。

### 4. 深隧寻珍

- **赛段要求：** 通过横向通道在三个竖向通道中寻找目标物体，过程中要避开障碍物。

**目标物体：**

1. 大桶可乐，识别后语音播报：**"识别到可乐瓶"**，击倒可乐瓶（可以使用身体任何部位）
2. 橙色球，识别后语音播报：**"识别到橙色小球"**，使小球明显晃动（不允许使用后空翻相关动作）
3. 足球，识别后语音播报：**"识别到足球"**，将小球踢入门内（可以连续撞击，可以使用身体任何部位）

**障碍物：**

1. 限高杆，识别后语音播报：**"识别到限高杆"**
2. 无法跨越障碍物，识别后语音播报：**"识别到无法跨越障碍"**

**避障行为：**

1. 需要从限高杆下通过，全程不允许碰撞限高杆
2. 需要绕过无法跨越障碍物，不允许碰撞，可以穿过虚线借道后再返回

**赛前准备：** 3 个目标物体和 3 个障碍物，在赛前均会做随机调整。

**竞赛规则：**

1. 跌倒或人工干预移动设备按评分标准扣分；
2. 未按规则与目标物体完成交互按评分标准扣分；
3. 语音播报错误或碰撞障碍物按评分标准扣分；
4. 未按规则识别障碍物并通过障碍物按评分标准扣分；
5. 未到达出口位置按评分标准扣分；
6. 放弃本赛段按评分标准扣分。

### 5. 孤梁稳渡

- **赛段要求：** 通过连续独木桥，穿过独木桥上的虚线后可以跳下。
  1. 必须是四个足底都越过虚线后才可以跳下
  2. 跳下时，不允许身体撞击独木桥
- **赛段结束：** 设备成功跳下，当前赛段结束。

**竞赛规则：**

1. 独木桥掉下或跌倒或人工干预移动设备按评分标准扣分；
2. 未按赛段要求跳下按评分标准扣分；
3. 未通过弯道或正常跳下按评分标准扣分；
4. 放弃本赛段按评分标准扣分。

### 6. 撷金建功

- **赛段要求：** 将足球从出口位置踢出，设备摔倒或者碰撞独木桥不扣分，小球从独木桥上飞出后不允许放回；
- **比赛结束：** 机器狗来到终点位置，四条腿的足底必须在圈内，趴下，比赛结束。

**竞赛规则：**

1. 四条腿的足底不在终点的圈内按评分标准扣分；
2. 足球飞出或者未正常踢出按评分标准扣分。

## 二、补充说明

1. 全程由机器狗自主完成比赛，设备启动后，不允许再次触碰电脑，触碰电脑则比赛结束；
2. 比赛过程中，可以使用笔记本电脑作为算力节点；
3. 比赛过程中，可以通过语音、触摸等方式控制机器狗继续比赛；
4. 所有虚线都可以踩线或跨越，所有实线不允许踩线和跨越；
5. 每次跌落或身体出线后，需要从跌落位置或当前赛段已经过的任意位置开始，中间已扣分数不再返还；
6. 跳过某一赛段，该赛段不得分，该赛段已扣分数不返还。
